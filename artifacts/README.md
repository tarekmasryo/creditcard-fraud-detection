# Artifacts

This directory is used by the notebook to export reusable model and threshold outputs.

These files are generated by running the notebook and are not committed to the repository by default.

Typical generated files may include:

- `xgb_pipe.joblib`
- `xgb_calibrated.joblib`
- `rf_pipe.joblib`
- `rf_calibrated.joblib`
- `thresholds.json`

The notebook keeps artifact saving disabled by default for Kaggle runs. Enable the artifact-saving flag in the notebook if you want to generate these files.
